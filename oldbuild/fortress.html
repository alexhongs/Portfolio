Fortress

Designer/Programmer

2-Dimensinoal single/multiplayer tank game with dynamic animations and inspiring graphics.

Implemented "Fortress" as an extension to the game "Fortress 2" by Cosmos Entertainment, with friendly tank looks.

Implementation


Movement

Uses the line between two pre-defined points on the terrain to update the unitâ€™s next location on map.


Bullet Collision

Uses the power and angle input from user to calculate the earliest position in time that touches any consecutive line segment between the pre-defined points.


Terrain Destruction

When the bullet actually get to the point of collision, delete all pre-defined points within the radius of explosion, creates new points along the curve, and changes alpha values of all pixels in the radius to make it look clear.
